# 🤖 AI Resume Screening Assistant

A multi-page production-ready recruitment screening tool built with **Streamlit** and powered by the ultra-fast **Groq AI Engine (`qwen-32b`)**. This application helps recruiters instantly parse resumes, match them against job roles, generate analytics, and manage applicant pipelines.
